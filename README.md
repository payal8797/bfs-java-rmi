# bfs-java-rmi

---Code files
- BFServer.java
- BFClient.java
- BFS.java
- BFSImpl.java

---Execution
Compile both server and client code by running commands.
=> javac BFClient.java
=> javac BFServer.java

Run Server and client code on the server machine.
Command for running
=> java BFClient
=> java BFServer
